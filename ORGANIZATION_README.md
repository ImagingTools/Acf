### ACF - Application Component Framework

[ACF (Application Component Framework)](https://github.com/ImagingTools/Acf) is our flagship project - a flexible and extensible framework for building component-based applications with comprehensive support for:

- **Component Architecture**: Flexible component-based design with dependency injection
- **Parameter Management**: Type-safe parameter handling and validation
- **Document Framework**: Template-based document management with undo/redo support
- **Serialization**: Built-in object persistence and serialization
- **2D/3D Graphics**: Utilities for geometric operations and visualization
- **Qt Integration**: Comprehensive Qt widgets and GUI components
